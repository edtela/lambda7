# The π-Space Motivation

## The Dimensional Collapse

Start with the volume of a 3D sphere:

```
n=3: V = (4/3) π r³
```

Take derivatives (dimensional collapse toward singularity):

```
n=3: (4/3) π r³   — volume (bulk)
n=2: 4π r²       — surface (shell)
n=1: 8π r        — circumference (line)
n=0: 8π          — point (source)
```

At the singularity, only **8π** remains — the seed constant.

This is also Einstein's gravitational constant: **8πG**

## The Dimensional Series

Build back up with factorials (from repeated integration):

```
n=0: 8π/0! × r⁰ = 8π
n=1: 8π/1! × r¹ = 8πr
n=2: 8π/2! × r² = 4πr²
n=3: 8π/3! × r³ = (4/3)πr³
...
```

Sum to infinity:

```
Σ (8π/n!) rⁿ = 8π × eʳ
```

**Assumption:** All dimensions at a singularity are circular (most symmetric geometry).

As r → 0: **8π × e⁰ = 8π** (singularity contains only the seed)

## Discrete vs Continuous

Set r = π. Compare two summation methods:

**Discrete (quantum):** Particles exist at integer dimensions n = 0, 1, 2, 3...
```
Σ (8π/n!) πⁿ = 8π × eᵖⁱ = 581.59
```

**Continuous (classical):** Smooth integral over all rotation angles
```
∫_{-π}^{π} 8π eʳ dr = 16π sinh(π) = 580.50
```

**The difference:**
```
8π eᵖⁱ − 16π sinh(π) = 8π e⁻ᵖⁱ
```

This is the **quantum tax** — what you pay for discreteness. The "antimatter shadow" left behind when going from discrete quantum dimensions to continuous classical rotation.

## The Floor/Ceiling Bridge

Scaling from π³ → π⁴ means multiplying by π ≈ 3.14

But dimensions are integers: 3, 4, 5...

When you "land" on a dimension, you must quantize:

```
floor(π) = 3
ceil(π)  = 4
```

**The floor/ceiling IS dimensional quantization.**

You can't have π dimensions — you snap to 3 or 4.

### Particle Coefficients

The same quantization applies to particle structure:

```
2π = 6.283...

floor(2π) = 6 → proton (closed rotation)
ceil(2π)  = 7 → strange baryons (next integer)
```

You can't have 2π quarks' worth of rotation — you snap to 6 or 7.

The strange quark "pushes" past the floor(2π) boundary to ceil(2π).
